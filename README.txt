This library implements a simple linked list

Files:
	Main:
		LinkedList.cpp
		LinkedList.h
	Test:
		Test.cpp

The Main directory contains the cpp and header file for the LinkedList class
	The header contains the class and function declarations
	The cpp contains the function definitions

The Test directory contains a cpp file that tests out the functionality of the LinkedList class as well as showing how it 
can be used.
